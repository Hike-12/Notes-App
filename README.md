
# Project Title

Notes App

This is a notes app with various features


## Tech Stack

**Client:** React,JSX,JavaScript,CSS

**Server:** Django

**Database:** SQLlite

## Features
1.Add Notes
2.Delete Notes
3.Edit notes
4.Rich Text Editor to help you format the notes with different syles (bold, italic, underlined, alignment and much more).
